# Quiz System (Django + Docker)

Интерактивная образовательная викторина в стиле **«Кто хочет стать миллионером»**, предназначенная для учебных занятий с детьми.

Система позволяет ученикам проходить тесты, соревноваться друг с другом и сохранять результаты.
Проект разработан с упором на **простоту использования для преподавателя** и **быстрый запуск на сервере**.

---

# Основные возможности

### Интерактивная викторина

* игровой интерфейс в стиле **Кто хочет стать миллионером**
* 3 подсказки - помощь друга, голосование зала, 50/50
* таймер **45 секунд на вопрос**
* перемешивание вопросов и вариантов ответа
* поддержка **тёмной и светлой темы**
* таблица рекордов

---

### Система обучения

Перед началом игры ученик выбирает:

1. группу
2. своё имя
3. урок

После этого запускается тест.

Результаты автоматически сохраняются и отображаются в **таблице рекордов**, что создаёт соревновательный эффект между учениками.

---

# Система сложности вопросов

Каждый урок должен содержать **15 вопросов**, разделённых по уровню сложности:

| Номер вопроса | Сложность |
| ------------- | --------- |
| 1–5           | Лёгкие    |
| 6–10          | Средние   |
| 11–15         | Сложные   |

При этом:

* вопросы внутри группы **перемешиваются**
* варианты ответа **перемешиваются**

Это делает прохождение теста уникальным каждый раз.

# Импорт вопросов из Excel

Для удобства преподавателя реализован **импорт из Excel** прямо в Django Admin.

Файл должен содержать **ровно 16 строк**:

* 1 строка — заголовок
* 15 строк — вопросы

Ограничения безопасности:

* максимум **300 KB**
* максимум **5 импортов в минуту**

---

### Формат Excel

| group | lesson | difficulty | question | option_a | option_b | option_c | option_d | correct |
| ----- | ------ | ---------- | -------- | -------- | -------- | -------- | -------- | ------- |

Пример:

| Суббота-11 | Интернет | 1 | Что такое фишинг? | Рыбалка | Кража паролей | Антивирус | Интернет | 2 |

# Технологии

Проект использует Docker и Docker Compose, разворачивая сразу Nginx, django, PostgreSQL.

Полный список технологий: 

* **Django**
* **Django REST Framework**
* **PostgreSQL**
* **Docker**
* **Nginx**
* **Gunicorn**

# Быстрый запуск через Docker

## 1. Клонировать репозиторий

```
git clone https://github.com/ADMania/Quiz-docker.git
cd Quiz-docker
```

---

## 2. Запустить контейнеры

```
docker compose up -d --build
```

Будут запущены:

* Django backend
* PostgreSQL
* Nginx

---

## 3. Выполнить миграции

```
docker compose exec backend python manage.py migrate
```

---

## 4. Создать администратора

```
docker compose exec backend python manage.py createsuperuser
```

---

## 5. Открыть проект

Frontend:

```
http://localhost:8080
```

Admin:

```
http://localhost:8080/programmist-admin
```
